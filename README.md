# 如何使用
运行train.py训练，使用NeuronChat.py则是使用训练好的模型

你可以选择直接运行NeuronChat.py使用以训练好的模型（best_model.pth），也可以选择运行train.py重新训练

请注意 use.py 已弃用，它可能无法运行
